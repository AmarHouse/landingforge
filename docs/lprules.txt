Se o objetivo é criar uma IA que gere landing pages de nível **Stripe, Apple, Linear, Vercel, Notion, Airbnb, Framer e Shopify**, então você precisa combinar dezenas de disciplinas diferentes. A maioria dos geradores atuais conhece apenas 10-20% delas.

As principais "leis" e frameworks de excelência são:

---

# 1. WCAG (Acessibilidade)

Base obrigatória.

Abrange:

* Contraste
* Navegação por teclado
* Leitores de tela
* Hierarquia semântica
* Estados de foco
* Alt text

Regras importantes:

AA:

* Texto normal ≥ 4.5:1
* Texto grande ≥ 3:1

AAA:

* Texto normal ≥ 7:1
* Texto grande ≥ 4.5:1

---

# 2. Teoria das Cores

Define:

* Harmonia
* Contraste
* Emoção
* Identidade

Sistemas:

### Monocromático

1 cor + tonalidades

### Complementar

Opostas no círculo cromático

### Análoga

Vizinhas

### Triádica

3 cores equidistantes

### Tetrádica

4 cores balanceadas

---

# 3. Gestalt

Talvez a teoria mais importante para UI.

Princípios:

### Proximidade

Elementos próximos parecem relacionados.

### Similaridade

Elementos parecidos são agrupados mentalmente.

### Continuidade

Olho segue caminhos naturais.

### Fechamento

Cérebro completa formas.

### Figura-Fundo

Separação clara entre foco e contexto.

---

# 4. Hierarquia Visual

Define a ordem de atenção.

Sequência ideal:

1. Headline
2. CTA
3. Benefício principal
4. Imagem
5. Prova social
6. Detalhes

Ferramentas:

* Tamanho
* Contraste
* Cor
* Espaço
* Posição

---

# 5. Princípio de Miller

Memória de curto prazo.

Usuário processa:

7 ± 2 itens

Melhor prática moderna:

3–5 itens por bloco.

---

# 6. Lei de Hick

Quanto mais opções:

Mais lenta a decisão.

Landing page:

Ideal:

1 CTA principal.

Ruim:

* Comprar
* Saber mais
* Download
* Ver vídeo
* Falar com vendas
* Comparar planos

Tudo junto.

---

# 7. Lei de Fitts

Alvos maiores são mais rápidos.

CTA:

Ideal:

44px–60px mínimo.

Mobile:

48px–64px.

---

# 8. Lei de Jakob

Usuários esperam padrões conhecidos.

Exemplos:

Logo → canto superior esquerdo

Menu → topo

CTA → hero

Não reinventar.

---

# 9. Lei de Tesler

Complexidade não desaparece.

Ela deve ser absorvida pelo sistema.

A landing page deve parecer simples mesmo que o produto seja complexo.

---

# 10. Lei de Prägnanz

Versão avançada da Gestalt.

O cérebro prefere:

* Simplicidade
* Simetria
* Estruturas previsíveis

---

# 11. Lei de Aesthetic-Usability

Algo bonito parece mais fácil de usar.

Visual premium:

Aumenta confiança.

Mesmo sem alterar funcionalidade.

---

# 12. Cognitive Load Theory

Carga cognitiva.

Remover:

* Ruído
* Elementos inúteis
* Texto excessivo

---

↓

1.333
1.618
